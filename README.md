# 胡希恕经方辨证系统

> 百分百模拟胡希恕先生辨证施治过程的本地化程序

## 简介

本系统基于胡希恕先生辨证体系，通过六经-方证-体质三层辨证，模拟胡希恕先生抓主症、辨六经、定方证、参体质的诊疗思路。

## 知识库

内置 30+ 胡希恕核心方剂：
- 太阳病方：桂枝汤、麻黄汤、葛根汤...
- 阳明病方：白虎汤、承气汤类...
- 少阳病方：小柴胡汤、大柴胡汤...
- 太阴病方：理中汤...
- 少阴病方：四逆汤、真武汤...
- 厥阴病方：乌梅丸...

## 快速开始

```bash
# 安装
./install.sh    # Mac/Linux
install.bat     # Windows

# 启动
./start.sh
```

访问 http://localhost:8501
