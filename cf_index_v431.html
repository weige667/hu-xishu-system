脉诊操作 (胡老规范) - 折叠说明 ↓

📖 展开/收起 脉诊操作流程说明
① 准备体位:
  1. 患者坐位或仰卧, 手臂自然舒展放平, 掌心向上
  2. 肘部与心脏保持同一水平, 腕下可垫脉枕
  3. 安静环境, 避开剧烈运动、饮酒、餐后
  4. 医生呼吸平稳, 宁神静气, 手暖
② 布指定位:
  1. 中指按高骨(桡骨茎突)内侧 = 关脉
  2. 食指(关前腕侧) = 寸脉
  3. 无名指(关后肘侧) = 尺脉
  4. 三指呈弓形, 指腹(螺纹面)着脉, 平齐
  5. 指距: 臂长疏、臂短密, 小儿一指定三关
③ 总按 (先看整体):
  1. 三指同时按三部 (寸关尺)
  2. 先浮取(轻) → 中取(不轻不重) → 沉取(重)
  3. 看三部脉象是否均匀、协调, 把握整体气血
  4. 确定整体脉象: 浮? 沉? 迟? 数? 弦? 弱?

⚠️ 胡老强调: 先总按看大势, 不要一开始就分部

④ 分部 (再看细节):
  1. 分别单独按每一部: 寸 → 关 → 尺
  2. 每部都做浮中沉三候 (九候)
  3. 细察某一脏腑的虚实, 看哪一部异常
  4. 口诀: 先总后单, 九候方彻
脏腑对应: 左手 寸(心) 关(肝) 尺(肾) | 右手 寸(肺) 关(脾) 尺(命门)

⑤ 腹诊 (胡老必做 - 与脉诊同等重要)

胡老腹诊操作流程:
  1. 患者仰卧, 两腿伸直, 双手放松; 医生坐或立于患者右侧
  2. 手法轻柔, 由浅入深; 全手掌轻按, 从无病部位开始, 渐及病所
  3. 顺序: 心下 → 胸胁 → 脐上 → 脐周 → 少腹 → 小腹
  4. 观察: 腹壁形态、张力、压痛、抵抗、搏动、振水音
