## 📋 Vercel 部署步骤

1. 访问 https://vercel.com/
2. 用 GitHub 账号注册（推荐）
3. 安装 Gitee 集成：
   - 访问 https://vercel.com/integrations/gitee
   - 点 Add Integration
   - 授权 Gitee 账号
4. 在 Vercel 创建项目：
   - Add New → Project
   - 选 wang730827/hu-xishu-system
   - Framework: Other
   - Install Command: pip install -r requirements.txt
5. 部署
6. 获得 URL: https://hu-xishu-system-xxx.vercel.app

## 📱 访问方式
