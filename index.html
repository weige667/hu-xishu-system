危证识别 (胡老强调"人命关天")

⚠️ 少阴危证 (四逆 + 脉微细)

特征: 四肢厥冷 + 脉微细 + 但欲寐 + 下利清谷

主方: 四逆汤 → 脉微欲绝转通脉四逆汤

胡老原话: "亡阳则死, 此证最险, 一逆尚引日, 再逆促命期"

⚠️ 厥阴危证 (寒热错杂 + 蛔厥)

特征: 消渴 + 气上撞心 + 心中疼热 + 饥而不欲食 + 食则吐蛔

主方: 乌梅丸

胡老原话: "寒热错杂, 上热下寒, 久则成劳"

⚠️ 急腹证 (大承气汤 / 大陷胸汤)

特征: 腹满痛拒按 + 潮热 + 谵语 + 大便不通 / 少腹硬满

主方: 大承气汤 (急下存阴) / 大陷胸汤

胡老案例: 70 多岁老妇人痢疾神昏谵语, 腹诊拒按 → 大承气汤一剂愈

⚠️ 大出血 / 脱证

特征: 吐血 / 衄血 / 便血 / 崩漏 + 面色苍白 + 脉芤/微细

主方: 泻心汤 (实热) / 黄土汤 (虚寒) / 独参汤 (气脱)

胡老原话: "急性失血, 中医有四法: 止血、消瘀、宁血、补虚"

⚠️ 阳虚外感 (麻黄附子细辛汤证)

特征: 无热恶寒 + 脉沉细 + 精神不振 (但欲寐)

主方: 麻黄附子细辛汤 / 麻黄附子甘草汤

胡老案例: 47 岁男, 感冒两天, 自觉无精神 + 两手逆冷 + 脉沉细 → 麻黄附子甘草汤加川芎

胡老辨证六步法 (官方整理版)

① 采集四诊

主动按诊断标准逐项排查, 不等病人诉说。例如发热: 有无恶寒? 身痛? 汗出? 鼻部症状?

② 辨病位

表 / 里 / 半表半里

③ 辨病性

阴 / 阳 (落实在寒热虚实)

④ 定六经

表阳 = 太阳 / 表阴 = 少阴 / 里阳 = 阳明 / 里阴 = 太阴 / 半阳 = 少阳 / 半阴 = 厥阴

⑤ 定治法

太阳 = 汗 / 阳明 = 清下 / 少阳 = 和 / 太阴 = 温 / 少阴 = 温 / 厥阴 = 寒热并用

⑥ 细辨方证

抓主症 → 定方 → 加减用药

脉诊操作流程 (胡老亲讲《讲仲景脉学》)

调息平脉

心中先有"平脉"标准 (老壮儿童、男女肥瘦、四时平脉均不同)

三部定位

中指按高骨 = 关, 食指(前) = 寸, 无名指(后) = 尺

三候取脉

浮取 (轻下指) / 中取 (不轻不重) / 沉取 (重下指) → 三部九候

循序细审

脉动 (位置+至数) / 脉体 (力量+广度) / 血行 (畅滞) → 逐一审查, 不可二用

太过与不及

太过 (浮数实大滑) → 阳热实; 不及 (沉迟虚细涩) → 阴寒虚

腹诊操作流程 (胡老临床必做)

体位

患者仰卧, 两腿伸直, 双手放松; 医生坐或立于患者右侧

手法

由浅入深, 由轻到重, 顺序: 心下 → 胸胁 → 脐上 → 脐周 → 少腹

六大腹证

① 心下痞 → 泻心/五苓; ② 胸胁苦满 → 柴胡; ③ 腹主动脉搏动 → 苓桂/桂甘; ④ 少腹不仁 → 肾气丸; ⑤ 腹皮拘急 → 建中; ⑥ 少腹硬满 → 抵当/陷胸
